# ✨ 크랙 문장 부풀리기 (Gemini)

[crack.wrtn.ai](https://crack.wrtn.ai) 캐릭터 채팅에서 **내가 짧게 쓴 대사·행동을 문학적으로 늘려주는** Tampermonkey 유저스크립트입니다. Gemini API를 사용하며, **유저(나) 본인**의 대사·행동만 풍부하게 써주고 **상대 주요 캐릭터는 건드리지 않습니다.**

## 🚀 설치

1. 브라우저에 [Tampermonkey](https://www.tampermonkey.net/) 확장 프로그램을 설치합니다.
2. 아래 버튼을 누르면 Tampermonkey 설치 화면이 뜹니다 → **설치** 클릭

   👉 **[클릭해서 설치하기](https://raw.githubusercontent.com/YOUR_ID/crack-sentence-expander/main/crack-sentence-expander.user.js)**

3. `crack.wrtn.ai` 새로고침하면 오른쪽에 패널이 떠요.

> 코드가 업데이트되면 Tampermonkey가 알아서 새 버전을 받아옵니다. ✅

## ⚙️ 처음 한 번 설정

1. 패널 오른쪽 위 **⚙️** 클릭
2. **Gemini API 키** 입력 ([Google AI Studio](https://aistudio.google.com/)에서 무료 발급)
3. **🔄 사용 가능한 모델 불러오기** → 드롭다운에서 모델 선택
4. (선택) **페르소나**·**캐릭터 이름** 입력
5. **저장**

## 🖱️ 사용법

- **💬 대사칸**: 내가 말하는 내용 → `"따옴표"`로 출력
- **🎬 행동칸**: 내가 하는 행동·의도 → `*이탤릭*` 지문으로 출력 (지시문을 명령 대사로 바꾸지 않고 *실제로 그 행동을 하는 장면*으로 묘사)
- **👤 시점**: 1인칭 / 3인칭 선택 (3인칭은 캐릭터 이름으로 서술)
- **길이**: 짧게 / 보통 / 길게
- **✨ 늘리기** (또는 Ctrl+Enter) → 결과 확인
- **💬 채팅창에 바로 넣기** / **📋 복사** / **🔄 다시 뽑기**

## ✨ 주요 기능

- 대사칸 / 행동칸 분리 입력
- 유저 페르소나 반영
- 1인칭 ↔ 3인칭 전환
- 3인칭에서 단역·배경 NPC 대사·묘사 허용 (**주요 캐릭터 제외**)
- 사용 가능한 Gemini 모델을 목록에서 선택
- 결과를 크랙 채팅창에 바로 입력
- 드래그 이동 + 위치 기억

## 🔒 개인정보

API 키는 **본인 브라우저에만** 저장되며 외부로 전송되지 않습니다 (Gemini API 호출 제외). 절대 다른 사람과 공유하지 마세요.
